C&S Connect Overlay

You can use this page to do the following:

  • Create a list of just your committees which will appear at the top of some large dropdowns on C&S Connect
  • Modify first consideration ballot defaults in the explanation field
  • Modify the appearance of title rows
  • Modify the appearance of input fields
  • Update User Info used for auto creating forms for you
  • Parse Minutes and create a table list of actions and motions
  • Create committee groups for ballots
